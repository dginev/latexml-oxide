Struct rtx_package::package::tokens::Tokens

source · 
pub struct Tokens(_);
Expand description

Tokens are a thin wrapper over a vector of Token objects usually read from a Mouth. They are usually treated as an immutable interface, an have to be consumed via .unlist() for access to the underlying data.
