Expand description

Perl-faithful body-digest for VBoxContents / HBoxContents parameters. Mirrors readBoxContents($gullet, $everybox, $mode) exactly: pushes a fresh frame for $mode, digests tokens until matching T_END, pops the frame.

The mode-aware variant matters when \vtop is invoked inside a p{} column body (alignment is in horizontal mode, but the \vtop’s VBox content must be digested in internal_vertical mode so \@startpbox’s \vtop\bgroup ... doesn’t try to close groups in the wrong mode). Witness: 2210.13325 \begin{tabular}{|p{1cm}|...} — pre-fix Rust inherited the surrounding horizontal mode and emitted \vtop errors; Perl uses ‘internal_vertical’ here regardless of where the \vtop was invoked.
