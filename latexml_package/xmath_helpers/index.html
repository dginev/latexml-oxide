Module xmath_helpers 

Source
Expand description

XMath helper functions for building XMDual token streams. Port of Perl’s I_dual, I_arg, I_xmarg, I_xmref, I_wrap from Base_XMath.pool.ltxml.

These functions generate token lists that expand into semantic math markup. Used by physics.sty, diffcoeff.sty, and other packages that need XMDual structures.
