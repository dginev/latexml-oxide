pub const BINDINGS: &[(&str, &str, BindingLoader)];
Expand description

Single source of truth for the compiled in-distro bindings. Each entry pairs a filename (with its .pool / .sty / .cls / .def extension) to its load_definitions fn. Used by dispatch (runtime loader) and by class_binding_names (prefix-match fallback in latexml_core::binding::content::load_class). Please keep this list as the ONLY place where binding filenames are enumerated — introducing a second list drifts quickly.
