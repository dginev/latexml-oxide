Perl finalize (Core/Document.pm L452, $node->removeAttribute($n) if $n =~ /^_/) drops every _-prefixed helper attribute — the ones holding fonts, source boxes, id bookkeeping — before anything is output. --xmath reads the digested document directly, upstream of our equivalent sweep (latexml_core::document L599), so run it here or _font="4568…" leaks into the output where Perl shows none.
