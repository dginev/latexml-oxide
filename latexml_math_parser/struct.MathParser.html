pub fn count_raw_trees(&mut self, input: &str) -> Option<usize>

Count the number of raw Marpa grammar trees for a lexeme string (for unit testing). This counts ALL derivation trees the grammar produces, before semantic pruning and deduplication. Useful for detecting grammar ambiguity. Returns the count, or None if recognition fails.
