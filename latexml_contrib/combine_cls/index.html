Stub for combine.cls (multi-document combiner).

combine.cls (TL latex/combine/combine.cls) is a class for merging multiple LaTeX documents into a single PDF. It defines {papers}, {document*}, etc. envs that wrap sub-document content. Our HTML/XML output is single-document; just render the body transparently. Witness 2310.14794.
