Module discard_env

Shared helper: discard an environment body by reading tokens up to matching \end{kind}. Mirrors the Perl discard_env_body closures used in ar5iv-bindings/nicematrix / forest / diagrams / pb-diagram stub packages.

Perl reference (identical across those stubs):

sub discard_env_body {
  my ($stomach, $kind) = @_;
  my $gullet = $stomach->getGullet;
  $stomach->bgroup;
  if (!$reported{$kind}) {
    $reported{$kind} = 1;
    Error('undefined', "{$kind}", $gullet,
      "$kind has no support, this is a stub binding."); }
  while (my $_ = $gullet->readUntil(T_CS('\end'))) {
    my $_drop_open = $gullet->readToken;
    my $env        = $gullet->readBalanced;
    last if ToString($env) eq $kind; }
  $stomach->egroup;
  return; }
