Module meta_class

Shared helper: the frontmatter interface common to the “addtolist ML meta-class” family (fairmeta / selfevolagent / openmoss and future siblings) — pre-print classes whose \author/\affiliation/\contribution/ \correspondence/\abstract/\beginappendix commands are defined on an \addtolist accumulator in the class BODY. An unknown .cls body is not raw-loaded (OmniBus extracts dependencies only), so every one of those is Error:undefined without a binding. Mirrors the shared-helper pattern of crate::discard_env.

install_meta_class_frontmatter
Install the identical, order-independent frontmatter routing shared by every sibling class: route \author to the creator sink and the rest to \@add@frontmatter/\lx@add@abstract so they reach <ltx:document> frontmatter. Call once from inside the class’s LoadDefinitions! block.
