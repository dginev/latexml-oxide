pub fn class_binding_names() -> &'static [&'static str]
Expand description

Names of all registered class bindings in this crate (without the .cls suffix), in registration order. Consumed alongside latexml_package::class_binding_names by the Perl-parity LoadClass prefix-match fallback, so contrib classes like memoir, siamltex, scrbook, etc. are eligible as alternates for unknown author-renamed classes. Lazily materialized from BINDINGS once and cached via OnceLock.
