Defines a macro, a binding analog to \def

A prototype will be parsed into a command sequence and a list of parameters. Any macro arguments will be substituted for parameter indicators (eg #1) in the Tokens or tokenized string and the result is used as the expansion of the control sequence. If a closure is used, it is called at expansion time and should return a list of tokens as its result.
