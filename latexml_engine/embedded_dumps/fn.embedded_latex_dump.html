pub fn embedded_latex_dump(prefer: Option<u32>) -> Option<&'static str>
Expand description

Bundled latex.YYYY.dump.txt content for the year that best matches prefer. See embedded_plain_dump for cache semantics. First call per machine-boot pays ~6.5 ms gunzip + ~5 ms write; every subsequent process across the boot reads the cached file in ~1 ms.
