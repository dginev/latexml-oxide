Stomach-level counterpart to read_box_contents.

Perl: readBoxContents calls $stomach->beginMode($mode), then reads/digests tokens Predigest box contents by invoking T_BEGIN, which triggers the stomach’s bgroup/egroup mechanism to properly handle the box body.

Perl’s List() simplification (List.pm line 41-44): When a vertical-mode List has exactly one non-empty item and that item’s mode is also vertical, return the item directly instead of wrapping in a List. This enables is_vbox property propagation for nested \vbox/\vtop.
