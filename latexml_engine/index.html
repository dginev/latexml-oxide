latexml_engine — TeX/LaTeX engine pool bindings (1:1 match with Perl LaTeXML/Engine/*.pool.ltxml). Split out of latexml_package to reduce CI compile-time RAM peaks and improve incremental rebuild locality. See docs/SYNC_STATUS.md “latexml_engine extraction” for the full motivation.

Loading hierarchy (mirrors Perl): LaTeX.pool → TeX.pool → Base.pool → Base_, TeX_, eTeX, pdfTeX, Base_Deprecated LoadFormat(‘plain’) → plain_bootstrap → plain_base → plain_dump → plain_constructs → math_common LoadFormat(‘latex’) → latex_bootstrap → latex_base → latex_dump → latex_constructs
