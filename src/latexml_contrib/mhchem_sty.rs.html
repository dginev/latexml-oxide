1//! mhchem.sty — chemical formula typesetting.
2//!
3//! TODO(strict-perl-parity): once `latexml_engine` can faithfully
4//! handle the expl3 / xparse / chemgreek raw-load chain (currently
5//! the gaps are around `\group_begin:` non-boxing-frame handling
6//! and l3regex/l3tl-analysis register access during the chemgreek
7//! load triggered by the first `\ce{...}`), DELETE this binding so
8//! that `\usepackage{mhchem}` raw-loads the actual TL `mhchem.sty`,
9//! matching Perl LaTeXML's behavior (Perl has no `mhchem.sty.ltxml`).
10//! Driver paper: arXiv:1806.06448 (3 errors → 0 errors with this
11//! stub; full chemistry rendering needs the engine fix).
12//!
13//! Perl LaTeXML has no `mhchem.sty.ltxml` and raw-loads the actual
14//! TL `mhchem.sty` (which `\RequirePackage{chemgreek}` →
15//! `\RequirePackage{xparse}` → heavy expl3 machinery). Perl's expl3
16//! emulation is mature enough that this works.
17//!
18//! Rust's expl3 emulation has gaps (e.g. `\group_begin:` non-boxing
19//! frame handling, `\l__tl_analysis_*_int` register access in
20//! l3regex/l3tl-analysis), so the chemgreek raw-load triggered by the
21//! first `\ce{...}` invocation leaves the gullet in an unbalanced state
22//! (open `\iffalse`, unmatched `{` at end-of-input).
23//!
24//! Until the expl3 cluster is fixed, this binding intercepts the
25//! mhchem load and provides a minimal stub: `\ce{...}` typesets its
26//! argument as roman text, no chemistry layout. This is a documented
27//! divergence from Perl LaTeXML — the full chemistry rendering needs
28//! a real port. Driver paper: 1806.06448 (3 errors → 0 errors).
29//!
30//! Stubs cover the public mhchem v3/v4 surface most papers actually
31//! use: `\ce`, `\cee`, `\cf`, plus `\mhchemoptions` (no-op).
32
33use latexml_package::prelude::*;
34
35#[rustfmt::skip]
36LoadDefinitions!({
37  // Accept both v3 and v4: the package option is `version=N` — handled
38  // at \usepackage time but irrelevant to our stub.
39  DefMacro!("\\mhchemoptions RequiredKeyVals", "");
40
41  // \ce{<formula>} — chemistry mode. Real mhchem renders subscripts,
42  // charges, arrows, etc. Papers invoke \ce{H_2O} / \ce{N_2} both in
43  // math context (equation*) AND in text context (paragraphs).
44  // \ensuremath wraps body in math mode if not already in math, so
45  // `_`/`^` parse as scripts in both contexts. Loses roman-text
46  // rendering of plain text chemistry, but avoids cascading errors.
47  // Witness: 0907.1390 (`\ce{N_2}, \ce{O_2}` in text → R=3→0).
48  DefMacro!("\\ce{}",  "\\ensuremath{#1}");
49  DefMacro!("\\cee{}", "\\ensuremath{#1}");
50  DefMacro!("\\cf{}",  "\\ensuremath{#1}");
51
52  // \arrow / \chemarrow — used inside \ce arguments. Stub as small text
53  // arrow so a `\ce{A \arrow B}` doesn't error if it leaks out.
54  DefMacro!("\\chemarrow", "\\rightarrow");
55});
