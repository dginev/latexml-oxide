1//! hpstatement.sty — the GHS "hazard" (H) and "precautionary" (P) statements
2//! for chemical labelling, part of the mhchem bundle. Provides
3//! `\hpstatement[..][..]{Hxxx}` (the statement sentence), `\hpnumber[..][..]{Hxxx}`
4//! (the formatted code), and `\hpsetup{keys}`.
5//!
6//! Perl LaTeXML ships no `hpstatement.sty.ltxml`; at the default `notex`
7//! setting it does not raw-load the real file either, so a
8//! `\usepackage{hpstatement}` document is parity (both error). This contrib
9//! binding goes beyond Perl by raw-loading the genuine TL `hpstatement.sty`.
10//! Note: hpstatement is an expl3 package (`\NewDocumentCommand`,
11//! `\keys_set:nn`) that pulls its sentence database from per-language data
12//! files (`hpstatement.inc/hpstatement-<lang>.inc.sty`) via `\file_input:n`.
13//! Raw-loading therefore exercises the engine's expl3 + `\file_input:n` paths;
14//! if a future TL layout hides those `.inc.sty` data files from kpathsea the
15//! statements degrade to their codes rather than erroring.
16use latexml_package::prelude::*;
17
18#[rustfmt::skip]
19LoadDefinitions!({
20  InputDefinitions!("hpstatement", noltxml => true, extension => Some(Cow::Borrowed("sty")));
21});
