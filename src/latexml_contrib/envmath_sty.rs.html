1//! envmath.sty (F. Bosisio) — extended math environments
2//! (`Equation`, `MultiLine`, `System`, `EQNarray`, `EqSystem`,
3//! `Cases`, ...).
4//!
5//! Raw-loading hits a `\catcode`\&=4` / `\OneShot@Amper`
6//! pattern that interacts badly with our alignment-template
7//! tracking — papers using envmath's auto-tab system run into
8//! a PushbackLimit infinite loop (witness arXiv:1501.05259:
9//! amsmath `cases` env + envmath redefining `&` → 650K-token
10//! pushback runaway). Perl LaTeXML skips raw-load
11//! (INCLUDE_STYLES=false), 10 warnings, conversion complete.
12//!
13//! Match Perl: stub the public API as no-ops. Map envmath's
14//! envs to their amsmath equivalents:
15//!   - `Equation` → `equation` (single-line equation)
16//!   - `MultiLine` → `multline*` (multi-line equation)
17//!   - `EQNarray` → `eqnarray` (alignment array)
18//!   - `System` / `EqSystem` → `align*` (system of equations)
19//!   - `Cases` → `cases` (case analysis)
20//!
21//! Lost fidelity: envmath's optional label arg and auto-numbering
22//! quirks. Gained: error-free conversion.
23use latexml_package::prelude::*;
24
25LoadDefinitions!({
26  Warn!(
27    "missing_file",
28    "envmath.sty",
29    "envmath.sty is minimally stubbed — extended math envs map to their amsmath equivalents."
30  );
31  // Top-level env aliases. Use Let! to bind both `\Foo` and
32  // `\endFoo` indirectly via the macro-expansion text.
33  DefMacro!("\\Equation", "\\begin{equation}");
34  DefMacro!("\\endEquation", "\\end{equation}");
35  DefMacro!("\\MultiLine", "\\begin{multline*}");
36  DefMacro!("\\endMultiLine", "\\end{multline*}");
37  DefMacro!("\\EQNarray", "\\begin{eqnarray}");
38  DefMacro!("\\endEQNarray", "\\end{eqnarray}");
39  DefMacro!("\\System", "\\begin{align*}");
40  DefMacro!("\\endSystem", "\\end{align*}");
41  DefMacro!("\\EqSystem []", "\\begin{align*}");
42  DefMacro!("\\endEqSystem", "\\end{align*}");
43  DefMacro!("\\Cases", "\\begin{cases}");
44  DefMacro!("\\endCases", "\\end{cases}");
45  // envmath's internal hooks — accept and ignore args.
46  def_macro_noop("\\StartMath@Err{}")?;
47  def_macro_noop("\\MakeAmper@Active{}")?;
48  def_macro_noop("\\MakeAmper@Tab")?;
49  def_macro_noop("\\OneShot@Amper{}{}")?;
50});
