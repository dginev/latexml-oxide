1//! titleref.sty — cross-reference titles (Donald Arseneau, 2001).
2//!
3//! titleref provides `\titleref{label}` to cross-reference the TITLE
4//! of a section/caption (rather than its number, like `\ref{label}`).
5//! Implementation details:
6//!  * Redefines `\label` to wrap `\@currentlabel` in a `\TR@TitleReference{<number>}{<title>}`
7//!    capture during `\edef`.
8//!  * Redefines `\@caption`, `\@sect`, etc. to also stash the title.
9//!
10//! Raw-loading titleref.sty into our engine is fragile: titleref's
11//! `\protected@edef\@currentlabel{\protect\TR@TitleReference
12//! {\@currentlabel}{\TR@currentTitle}}` interacts badly with our
13//! `\caption`/`\label` machinery — `\TR@currentTitle` may be unset and
14//! `\@currentlabel` may contain `\the<counter>` tokens that the edef
15//! processes incorrectly, surfacing "You can't use `}` after `\the`"
16//! cascades that swamp `\caption` (driver: 1103.2227).
17//!
18//! Provide a minimal stub that:
19//!  * Defines `\titleref` as a thin alias for `\ref` (the next-best behavior for HTML output — we
20//!    lose the actual section-title text but produce a working cross-reference).
21//!  * Stubs `\theTitleReference`, `\currenttitle`, and the package options as harmless no-ops.
22//!  * Does NOT redefine `\label`, `\@caption`, or any kernel sectioning command.
23//!
24//! Matches Perl's effective behavior: Perl LaTeXML also has no
25//! titleref binding; with the default `INCLUDE_STYLES=false` Perl
26//! emits "missing binding" and the raw `.sty` is not loaded. Our stub
27//! produces equivalent semantics (cross-refs work, title text is the
28//! number).
29use latexml_package::prelude::*;
30
31LoadDefinitions!({
32  Warn!(
33    "missing_file",
34    "titleref.sty",
35    "titleref.sty is minimally stubbed — \\titleref behaves like \\ref (number, not title)."
36  );
37  // `\titleref{label}` — fall back to `\ref{label}` so cross-refs work.
38  DefMacro!("\\titleref{}", "\\ref{#1}");
39  // `\theTitleReference{num}{title}` — formatting helper; default shows title.
40  // Kept as a 2-arg passthrough that returns the title (Perl's titleref
41  // default).
42  DefMacro!("\\theTitleReference{}{}", "#2");
43  // `\currenttitle` — title of current section; we don't track titles,
44  // so emit a noop placeholder.
45  def_macro_noop("\\currenttitle")?;
46  // No-op the option machinery. titleref accepts `[usetoc]` and
47  // `[nostar]` options that toggle implementation strategies; both
48  // become irrelevant with our stub.
49});
