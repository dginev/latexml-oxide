1//! No-op stub for constants.sty.
2//!
3//! constants.sty (TL — "Labeling and numbering constants") provides
4//! `\newconstantfamily`, `\C`, `\Cl`, `\refconstant`, `\resetconstant`.
5//! Its end-of-document bookkeeping (constants.sty L143-148) does a *raw*
6//! `\if@filesw … \makeatletter \input\jobname.aux \fi` to re-read the
7//! label `.aux` for the "constants may have changed, rerun" check.
8//!
9//! Our engine never opens `\@mainaux` at *runtime* (the kernel's
10//! `\openout\@mainaux\jobname.aux` is baked into the dump at build
11//! time, with the build-time jobname), so no in-memory `<jobname>.aux`
12//! buffer exists; the raw `\input\jobname.aux` then fails with
13//! `Error:missing_file:<jobname>.aux`. (Unlike LaTeX's silent `\@input`,
14//! constants.sty uses the hard `\input`.)
15//!
16//! Perl LaTeXML never observes this: its default TEXINPUTS excludes
17//! `/usr/share/texlive`, so constants.sty is reported missing-file and
18//! skipped — providing ZERO constants functionality. Verified on
19//! arXiv:2002.05335: Perl emits `Warning:missing_file:constants` and
20//! completes (its residual errors are unrelated math-mode issues).
21//!
22//! Match Perl's effective behavior with a no-op stub (RequirePackage
23//! the real `keyval` dep; no-op the user-facing constants API). All 70
24//! R-stage papers blocked on `<jobname>.aux` were checked: every one is
25//! CONVERR_1 (the single `.aux` error) and NONE invoke `\C`/`\Cl`/
26//! `\newconstantfamily`, so the stub costs no document content.
27//!
28//! Witnesses (CONVERR_1 → OK): 2004.04403, 2002.05335, 1501.05396,
29//! 1503.01423, 1505.03199, 1506.02544, 1506.03382, 1506.03705,
30//! 1508.03002, 1509.02505, 1512.06906, 1601.01096, 1603.05947,
31//! 1606.09238, … (70 total).
32
33use crate::prelude::*;
34
35#[rustfmt::skip]
36LoadDefinitions!({
37  RequirePackage!("keyval");
38  // User-facing constants API — no-op (matches Perl's missing-file
39  // skip; no observed paper uses these).
40  def_macro_noop("\\newconstantfamily{}{}")?;
41  def_macro_noop("\\renewconstantfamily{}{}")?;
42  def_macro_noop("\\resetconstant[]")?;
43  def_macro_noop("\\C[]")?;
44  def_macro_noop("\\Cl[]{}")?;
45  def_macro_noop("\\refconstant{}")?;
46});
