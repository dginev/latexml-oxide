3// fvextra.sty — extends fancyvrb (breaklines, breakanywhere, improved line
4// numbering, math-mode verbatim, …). Perl LaTeXML ships no fvextra.sty.ltxml
5// (it raw-loads the real file only under `--includestyles`; by default the
6// package is simply missing there). We raw-load unconditionally — house
7// idiom — so every environment/key a document declares
8// (`\DefineVerbatimEnvironment{Prompt}{Verbatim}{breaklines,…}`) is defined.
9//
10// AFTER loading, we neutralise ONLY fvextra's char-by-char break scanner:
11// `breakanywhere`/`breakbefore`/`breakafter` do `\let\FancyVerbBreakStart
12// \FV@Break` at key-SET time — a recursive scanner that measures every
13// character by boxing line-prefixes (`\sbox{\FV@LineBox}{\FV@BProcessLine
14// {#1}}`). In our engine that recurses through
15// `predigest_box_contents_in_mode` and grows the gullet pushback unboundedly
16// until the 650000 `PushbackLimit` Fatal fires (display path; 121/185 fatal
17// papers in sandbox-arxiv-2605, witness 2605.01024) or hangs the inline
18// `\Verb` path to Fatal:Timeout:TokenLimit — where Perl converts cleanly.
19// Aliasing the TARGET `\FV@Break` to `\relax` makes every later key-set
20// propagate `\relax`, and BOTH consumers gate on
21// `\ifx\FancyVerbBreakStart\relax`, taking their plain paths.
22//
23// The `breaklines` line-processor `\FV@ListProcessLine@Break` itself is
24// left INTACT: with the scanner relaxed it typesets an over-wide line as a
25// `\parbox[t]{\FV@LineWidth}` with ragged-right and breakable spaces —
26// plain TeX paragraph machinery our engine wraps natively, so the measured
27// height budget counts the same wrapped lines pdflatex produces and the
28// content stays inside the drawn frame (2605.00468 prompt boxes poked
29// 4-60px past the right border when this was over-neutralised to the
30// `@NoBreak` processor, which hboxes each SOURCE line unbroken).
31#[rustfmt::skip]
32LoadDefinitions!({
33  InputDefinitions!("fvextra", noltxml => true, extension => Some(Cow::Borrowed("sty")));
34
35  RawTeX!(r"\let\FV@Break\relax");
36});
