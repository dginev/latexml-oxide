8  // with latexml's subroutine macro definitions, so default it to *true*
9  // to avoid needless warnings. Perl adds `locked => 1` — critical here

11  // filehook.sty which itself defines \filehook@cmp; without the lock,

15  // Defensive stubs for filehook-2020/filehook-2019 commands. The raw
16  // filehook.sty selects one of these sub-files via

18  // Our search-paths-only find_file can't locate the versioned sub-files
19  // in TL (they're not in user paths), so the load falls back to
20  // filehook.sty itself — leaving the hooks undefined. Pre-define them
