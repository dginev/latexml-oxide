1//! mathenv.sty — TWO different packages ship under this name; the binding
2//! raw-loads the self-contained one and no-ops the mdwtools one.
3//!
4//! * **Bosisio's "Extended math environments"** (Francesco Bosisio, 1997) —
5//!   self-contained, defines `\newenvironment{EqSystem}`/`{Equation}` and an
6//!   active-`&` system-of-equations machinery. NOT in TeX Live (papers ship it
7//!   locally). Perl LaTeXML has no `mathenv.sty.ltxml`, so under the ar5iv gate
8//!   (`INCLUDE_STYLES=true`) it RAW-LOADS this file and converts cleanly
9//!   (witness 1710.07800 → 92 errors here without the raw-load, 0 with it).
10//!
11//! * **Mark Wooding's mdwtools `mathenv`** — opens with `\RequirePackage{mdwtab}`
12//!   (a full `tabular` rewrite). Our `mdwtab` stub can't supply its `\tab@*`
13//!   internals, so raw-loading Wooding's mathenv cascades (`\tab@setstate` /
14//!   `\tab@preamble` / … undefined → ~42 errors). A no-op stub avoids that
15//!   (witness 0910.3293). The principled fix for THIS variant is a working
16//!   `mdwtab` port; until then we keep the stub for it.
17//!
18//! Distinguish the two by the `mdwtab` requirement (Bosisio's file never
19//! mentions it; Wooding's does `\RequirePackage{mdwtab}`): raw-load the
20//! self-contained Bosisio variant like Perl, no-op the mdwtools one. Both then
21//! match Perl. (Task #273 — prefer raw-load over stubs; here the stub is kept
22//! ONLY for the variant whose dependency we can't yet raw-load.)
23use crate::prelude::*;
24
25LoadDefinitions!({
26  let requires_mdwtab = find_file(
27    "mathenv",
28    Some(FindFileOptions { ext_type: Some(Cow::Borrowed("sty")), ..Default::default() }),
29  )
30  .and_then(|p| std::fs::read_to_string(&p).ok())
31  .is_some_and(|content| content.contains("mdwtab"));
32  if !requires_mdwtab {
33    // Bosisio's self-contained mathenv → raw-load it, matching Perl.
34    InputDefinitions!("mathenv", noltxml => true, extension => Some(Cow::Borrowed("sty")));
35  }
36  // else: Wooding's mdwtools mathenv → no-op (avoid the broken-mdwtab cascade).
37});
