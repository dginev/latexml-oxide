23  // icml2018.sty L709: `\long\def\comment#1{}` — a review-annotation macro that
24  // gobbles its argument (authors write `\comment{\ref{…}}` to hide draft
25  // notes). Our binding intercepts icml2016/2017/2018 but had omitted it, so a
26  // paper using `\comment{…}` (via arxiv.sty → `\RequirePackage{icml2018}`) hit
27  // `undefined:\comment` where Perl (which loads icml2018) is clean. Gobbling
28  // is the intended, source-faithful behavior. Witness 1803.00942.
