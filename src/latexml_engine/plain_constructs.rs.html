15  // \#, \&, \%, \$, \_ math/text dispatch family — relocated from
16  // plain_base.rs to here so the dump path (where plain_base is skipped)
17  // also gets the dispatch macros and their `\lx@(text|math)@*` targets.
18  // Without this, dump's CharDef-38 for `\&` survives and breaks math-
19  // mode `$\&\&$` (it injects ALIGN-catcode `&` into the math parser).
20  // Mirrors plain_base.pool.ltxml:L70-77 (Perl uses Box-dispatching
21  // DefPrimitives — Rust's explicit math/text split is the WISDOM #44
22  // documented divergence). Rusts the dispatch via `\ifmmode` and routes
23  // math-mode through DefMath entries with proper role/meaning, text-
24  // mode through DefPrimitive emitting the literal char.
25  // `protected => true`: keep these dispatch macros UNEXPANDED under
26  // partial expansion (`\write`'s `XGeneralText`, `\edef`-without-
27  // `\protected@edef`, etc.). Without this, `\write {\&}` partial-
28  // expands `\&` → `\lx@text@amp` and `untex` serializes the
29  // internal CS literally — when `\input` re-reads with `@` in
30  // OTHER catcode, `\lx@text@amp` splits to `\lx` + `@text@amp`,
31  // i.e. `Error:undefined:\lx`. Perl avoids this because Perl's
32  // `\&` is a single `DefPrimitive`, not a dispatch macro; the
33  // WISDOM #44 documented Rust divergence makes us split at gullet
34  // time, so we need the protected flag to restore the round-trip
35  // semantics. Witness: hep-th9306154 / hep-ph9803499 /
36  // hep-th9203004 (harvmac `\listrefs` writing `\&` in references).
37  // `\#` as a single mode-aware PRIMITIVE (Perl's Box-dispatching DefPrimitive),
38  // not a protected dispatch MACRO. A primitive is non-expandable (survives the
39  // `\write`/`\input` round-trip — harvmac) AND `\ifx`-stable, so a paper using
40  // `\#` as a macro-stack sentinel (algochl.sty's `\alg@push`/`\alg@pop` —
41  // `\xdef\alg@Sc{\#}` then `\ifx`-compares the popped top against `\#`) works:
42  // the prior protected dispatch macro was NOT `\ifx`-stable in that context.
43  // Routes by IN_MATH to the existing math/text helpers at digest time,
44  // preserving rendering (and avoiding the dump CharDef's dropped-`#` in math).
45  // Witness 1811.00200 (llncs + algochl.sty). WISDOM #44.
46  DefPrimitive!("\\#", {

48      T_CS!("\\lx@math@hash") } else { T_CS!("\\lx@text@hash") };

54  // role/meaning, so math-mode `$\&\&$` still routes through `\lx@math@amp`
55  // rather than injecting a catcode-4 `&` — the breakage the prior override
56  // guarded against.)

558            }
559          }
