% infinitely through predigest_box_contents (every character measured by
% boxing a line-prefix), growing the gullet pushback until the 650000
% Timeout/PushbackLimit Fatal fired. Perl converts this cleanly.
% The fvextra binding routes the breaking line-processor to the
% non-breaking one, so the verbatim completes (font="typewriter" preserved)
% with browser-handled wrapping — output-faithful to Perl's HTML.
% Drove 119/121 fatal papers in sandbox-arxiv-2605 (witness 2605.01024).

\usepackage{fancyvrb}
\usepackage{fvextra}

\begin{document}
\begin{Prompt}

This is a deliberately long second line so the break machinery would engage repeatedly without terminating in the unfixed engine.
\end{Prompt}
\end{document}
